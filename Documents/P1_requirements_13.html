Purpose and goals

To attract new customers and inform our current customers relevant information about the restaurant.

Audience

Hungry people and families. The reason for this demographic can be explained due to a customers need to know the price and content of a given product provided by the restaurant. Important information such as allergies can also be found on such websites.

Content and organization

Necessary pages and features
  • About page
  • Menu page
  • Contact us page
  • Delivery time & address
  • Opening hours

Extra
  • Dish info pop up
  • A referral function

Requirements

Functional requirements
  • The site should have a working "Contact Us" form that will send the restaurant management an email
  • The site should inform the user about the restaurant menu's options via a list or a table
  • The site should inform customers about the restaurant's opening and closing hours via a table
  • The site should make the user feel warm and welcome like the restaurant does

Non-functional requirements
  • The user should be able to find the menu item they are looking for within 15 seconds from entering the website
  • The user should easily, and at a glance, be able to find any allergens in any menu item
  • Every page must load in less than one second

Final location

The website will be hosted on Github pages or Vercel
